🛒 E-Commerce Sales Analysis Dashboard | Power BI

📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze e-commerce sales performance. It provides valuable insights into sales trends, customer behavior, product performance, and profitability, helping businesses make data-driven decisions.

🚀 Key Features

* Interactive dashboard with slicers and filters
* Sales analysis by region, category, and sub-category
* Customer and product performance analysis
* Monthly and yearly sales trend visualization
* Profit and sales comparison
* Top-selling products and top customers
* KPI cards for Total Sales, Total Profit, Orders, and Average Order Value

🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Calculated Columns & Measures)
* Excel/CSV (Dataset)

📊 Dashboard Insights

* Identified top-performing product categories.
* Analyzed regional sales and profit distribution.
* Tracked monthly sales trends to identify seasonal patterns.
* Evaluated customer purchasing behavior.
* Monitored key business KPIs for better decision-making.
